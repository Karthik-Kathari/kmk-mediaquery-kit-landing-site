kmk-mediaquery-kit

A simple CLI tool to automatically scaffold responsive media queries for your plain HTML and CSS projects.

View on GitHub
Terminal
$ npm install -g kmk-mediaquery-kit
$ kmk-mq-kit
✓ Media queries scaffolded successfully!

Powerful Features

Everything you need to create responsive designs effortlessly

Auto-Detection

Automatically detects all CSS classes used in your HTML files and checks which ones are styled in your CSS.

Smart Scaffolding

Generates media query blocks across configured breakpoints without overwriting existing styles.

Watch Mode

Real-time updates with watch mode that monitors your files and automatically generates responsive styles.

Configurable

Fully configurable breakpoints and paths via JSON config. Customize to fit your project structure.

Lightweight

Minimal footprint and easy integration. Perfect for static web projects without heavy frameworks.

Non-Destructive

Appends new responsive styles without overwriting existing ones. Your current styles remain intact.

Quick Installation

Get started in seconds with a simple npm command

npm install -g kmk-mediaquery-kit

How to Use

Simple commands for powerful responsive scaffolding

CLI Commands

# Generate media queries once
kmk-mq-kit

# Watch for changes and auto-generate
kmk-mq-kit --watch

Configuration File

Create a mqscaffold.config.json file in your project root:

mqscaffold.config.json 
{
  "breakpoints": {
    "mobile": "768px",
    "tablet": "1024px", 
    "desktop": "1200px"
  },
  "cssPath": "styles/styles.css",
  "htmlPaths": ["index.html", "pages/*.html"],
  "outputPath": "styles/media-queries.css"
}

Recommended Project Structure

📁 your-project/
📄 index.html
⚙️ mqscaffold.config.json
📁 styles/
🎨 styles.css
📱 media-queries.css generated

HTML Integration

Link both CSS files in your HTML:

index.html 
<head>
  <link rel="stylesheet" href="styles/styles.css">
  <link rel="stylesheet" href="styles/media-queries.css">
</head>
